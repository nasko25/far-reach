# Far Reach Project

Far-Reach is a Shopify app that connects merchants to the Farcaster ecosystem via affiliates, who sell products to their social graph using Frames. Think Shopify Collabs but for Farcaster and on-chain

## Technologies Used
- React
- GraphQL
- TheGraph
- Airstack
- Privy
- Frames.js
- Base Sepolia
- Smart Wallet
- Gadget
- Shopify
